# Bakery Website

This is a simple bakery website built with React and TypeScript.
View the live demo [here](https://smokydabear.github.io/remmys-bakery-site/)

## Features

- Browse bakery items
- View item details
- Add items to cart
- Checkout process

## About

- Built with React and TypeScript for a modern web experience.
- Responsive design for mobile and desktop users.
- Easy to navigate and user-friendly interface.
- Styled with CSS for a visually appealing and modern aesthetic.
